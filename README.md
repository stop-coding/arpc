# arpc
  基于XIO实现的远程通信框架，用于分布式文件系统数据传输

#                    build
【说明】编译工程采用cmake编译

cd code root;
mkdir build
cd build
cmake ../    (default:debug mode)
make -j
make install


#                    file tree
* demo(demo测试应用)

* open_src (开源代码xio)

* src (arpc 源码)

* inc (arpc api)

# 其它说明
生成库文件libarpc.so
